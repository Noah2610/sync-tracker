# sync-tracker
Collaborative, live music editor.  
_Work in progress..._

## Development Setup
First, install dependencies:
```
yarn install
```

Then you need to run both the development server and the app.  
Run the development WebSocket server:
```
yarn dev:server
```

Run the frontend app:
```
yarn dev
```
